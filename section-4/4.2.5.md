# 4.2.5.  派生类型（typedef）

`YANG`可以使用“`typedef`”语句从基本类型定义派生类型。 基本类型可以是内置类型或派生类型，允许派生类型的层次结构。

派生类型可以用作“类型”语句的参数。

`YANG`示例：

```YANG
typedef percent {
  type uint8 {
    range "0 .. 100";
  }
}

leaf completed {
  type percent;
}
```

`XML`编码示例：

```xml
<completed>20</completed>
```
